# celebal-week5
Deep Learning Text Generation Project

## Text Generation using Vanilla RNN, LSTM, and GRU

### Objective
This project demonstrates how recurrent neural network architectures learn:

- Grammar and sentence structure
- Contextual dependencies
- Sequential patterns in text
- Next-word prediction
- Text generation

Three architectures are implemented and compared:

1. Vanilla RNN
2. LSTM (Long Short-Term Memory)
3. GRU (Gated Recurrent Unit)

The models are trained on the same corpus and evaluated using training loss and generated text quality.
